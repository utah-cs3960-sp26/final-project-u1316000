# Ideas Scratchpad

This file is a loose shared notebook for future story possibilities.

Purpose:
- quick scene ideas
- future location ideas
- future character ideas
- possible plotlines
- fun surreal set pieces
- half-formed continuity thoughts
- things a worker or human wants to remember without making them canon yet

This file is intentionally informal.
- It is okay to be rough.
- It is okay to leave fragments.
- It is okay to suggest something and never use it.
- If you are a worker appending new ideas, read the current file first and add genuinely new ideas rather than repeating or lightly remixing existing examples.

Use this file when:
- you have a fun idea for later
- you want to remember a possible escalation
- you want to note a future character, place, or set piece
- you want a human-friendly note that is easy to skim and edit by hand

Do not treat anything here as automatic canon.
- Canon happens in scenes, entities, hooks, facts, and assets after it is actually introduced.

How this differs from `story_direction_notes`:
- `IDEAS.md` is the loose human-readable scratchpad
- `story_direction_notes` are structured planning notes in SQLite for worker/runtime use

Suggested format:

## Event or encounter Ideas

- Transit crisis / tram robbery:
  A later tram ride could erupt into a robbery, transit strike, or route seizure that introduces new characters and turns the tram network into a more active plotline.

- Goose-back courier route:
  A sky route run by geese could become a reusable traversal layer, with whistle-based pickup points and weather-related complications.

- Name bureaucracy:
  Lost Property Window might eventually connect to a larger bureaucracy of misplaced names, swapped destinies, and bodies filed under the wrong heading.

## Character Seeds

- A nervous duck herald who announces disasters too early.
- A cabin-dwelling frog scholar who studies impolite hats.
- A man with 5 thumbs on his right hand (could be used to explore origin of main character's 5 thumbed left hand)

## Location Seeds

- Yesterday's Orchard:
  A place the tram can reach where fruit ripens backward and dropped apples return to the branch.

- The Apology Bridge:
  A bridge that only stabilizes when both sides admit what they actually want.

- A steampunk cottage nestled within a dense jungle

## Notes

- Keep ideas readable and concrete.
- Prefer one or two clear sentences over vague mood-only fragments.
- If an idea starts shaping real future continuity, it can also be copied into `story_direction_notes`.
- Treat the example seeds in this file as examples, not defaults to repeat verbatim.
